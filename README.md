## 📖 Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/tu-repo.git
   ```
2. Navega al proyecto:
   ```bash
   cd PRACTICA-FORMATIVA-OBLIGATORIA-GIT
   ```
3. Revisa las ramas:
   ```bash
   git branch -a
   ```
4. Cambia a la rama de tu interés:
   ```bash
   git checkout feature/perfil-nuevo-ui
   ```
